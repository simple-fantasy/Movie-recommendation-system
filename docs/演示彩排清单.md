## 6. 常见故障与应急
- 404 `/api/metrics/offline`：先运行 `python -m backend.scripts.evaluate_itemcf`
- 404 `/api/metrics/evaluation`：先运行 `python -m backend.scripts.evaluate_models --models all --ablation --k 10`
- NCF 未加载：确认 `ncf.pt` 与 `ncf_meta.json` 存在
- `ModuleNotFoundError`：确认已在正确环境并安装依赖
